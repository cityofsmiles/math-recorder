# Math Recorder by Jonathan R. Bacolod

## How to Use
1. I-copy ang link at i-open sa Chrome. Mag-login gamit ang sinend kong password.
![Login](https://res.cloudinary.com/dadubemrv/image/upload/v1769399056/j8vexqjl3077zjz9ywqs.jpg)

2. I-type ang infos.
![Infos](https://res.cloudinary.com/dadubemrv/image/upload/v1769398383/ce5kma5um9p9f17m1kjw.jpg)

3. Ilagay ang P.E. number, Activity number, at Quiz number. Halimbawa, 3.1, 3.2, etc.
![PE infos](https://res.cloudinary.com/dadubemrv/image/upload/v1769398521/aitbpxckgkb7f9fnjhiz.jpg)

![Quiz infos](https://res.cloudinary.com/dadubemrv/image/upload/v1769398606/f0rzxiexdi8kc2jpxf1t.jpg)

4. Ilagay ang scores. Iclick ang Save button everytime mag-eedit ng scores.
![Act scores](https://res.cloudinary.com/dadubemrv/image/upload/v1769398717/xjc9feaw2jkqra6lrel1.jpg)

![PE scores](https://res.cloudinary.com/dadubemrv/image/upload/v1769398776/mp40ggutjcuuspn60zhi.jpg)

![Quiz score](https://res.cloudinary.com/dadubemrv/image/upload/v1769398874/rssxzfhsp3igcjflovaa.jpg)

![F scores](https://res.cloudinary.com/dadubemrv/image/upload/v1769398919/imdxymzgunuceu0umatu.jpg)

5. Iclick ang Clear button kung gusto burahin lahat ng data sa table.
